# Portfolio Website

This is my personal portfolio website built using Next.js and Tailwind CSS.

## Live Demo

Check out the live demo of the website: [Link to Live Demo](https://alexvon.dev)

## Technologies Used

- Next.js
- Typescript
- Tailwind CSS
- Radix UI / Shadcn/ui

## Installation

To run this project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/alex-von/portfolio.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser and visit: `http://localhost:3000`
